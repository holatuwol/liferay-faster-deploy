Via Hosts File

Add the following to your hosts file to access the running servers.

PUBLIC_IPmaster.lrsupport.commaster
PUBLIC_IP74x.lrsupport.com74x
PUBLIC_IP73x.lrsupport.com73x
PUBLIC_IP72x.lrsupport.com72x
PUBLIC_IP71x.lrsupport.com71x
PUBLIC_IP70x.lrsupport.com70x
PUBLIC_IP62x.lrsupport.com62x
PUBLIC_IP61x.lrsupport.com61x
PUBLIC_IPdebug.lrsupport.comdebug
