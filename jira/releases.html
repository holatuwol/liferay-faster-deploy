Flattened Jira Release Notes

This is a prototype of what Liferay release notes might look like if we included the ticket descriptions on Jira. Users who view this might also be interested in the official Breaking Changes documentation found on GitHub. This page was built by pulling each of the releases listed on the PUBLIC - Liferay Portal Community Edition Releases page and is updated on an as-needed basis for demo purposes only.
