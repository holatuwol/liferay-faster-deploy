Flattened Jira Release Notes

This is a prototype of what Liferay DXP Release Notes might look like if we included the ticket descriptions on Jira. Users who view this might also be interested in the official Breaking Changes documentation found on GitHub.

This page was built by pulling each of the releases listed on the PUBLIC - Liferay Portal Community Edition Releases page and the PUBLIC - Liferay Product Delivery Releases page, as well as checking the 7.4 fixed version, which is included in the field cf[10210] (for updates, it is the update number; for quarterly releases, it is a bit more complicated).

This page is updated on an as-needed basis for demo purposes only.

Warning: This is NOT an official Liferay tool, nor is it supported by Liferay in any way.
